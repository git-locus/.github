# Security Policy

## Reporting a Vulnerability

**Non aprire una issue pubblica per vulnerabilità di sicurezza.**

Segnala le vulnerabilità privatamente tramite [GitHub Private Vulnerability Reporting](https://github.com/git-locus/.github/security/advisories/new) oppure contatta direttamente i maintainer.

Risponderemo entro 72 ore con una valutazione e i passi successivi.

## Scope

Fanno parte dello scope: `api`, `client`, infrastruttura (`docker2azure4student`).
